# Hyprland Dotfiles

Конфигурационные файлы для Hyprland системы на Arch Linux.

## Структура

- `hypr/` - Конфиг Hyprland (WM)
- `ags/` - Верхняя панель (GTK приложение)
- `waybar/` - Нижняя панель
- `rofi/` - App launcher
- `kitty/` - Терминал
- `swaync/` - Уведомления
- `gtk-3.0/` / `gtk-4.0/` - GTK темы
- `bin/` - Скрипты

## Документация

Подробное описание всех файлов и директорий доступно в [DOCUMENTATION.md](DOCUMENTATION.md).

## Установка

Для установки конфигурации используйте скрипт `install.sh`. Он автоматически создаст символические ссылки на конфигурационные файлы, установит скрипты в `~/.local/bin/` и сделает резервные копии существующих файлов.

```bash
chmod +x install.sh
./install.sh
```
